clickhouse 多分片集群搭建

ck分布式集群，采用本地表、分布式表 形式实现。本地表作为分片，分布式表作为数据集合。对于分布式表的插入会随机分配到配置的分片上，查询则会得到所有的数据集合。

本地表采用 Replicated 引擎，实现主备之间的数据同步，可以参考上篇文章 clickhouse单分片三副本高可用搭建，分布表采用 Distributed 引擎。
